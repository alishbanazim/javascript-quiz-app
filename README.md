# javascript quiz app

A responsive JavaScript Quiz Application built with HTML, CSS, and Vanilla JavaScript.

Users have **60 seconds** to answer **10 multiple-choice JavaScript questions**. The quiz provides instant feedback, automatically highlights the correct answer, and displays a personalized result screen based on the final score.

##  Live Demo

🔗 https://alishbanazim.github.io/javascript-quiz-app/

##  Features

-  10 JavaScript multiple-choice questions
-  60-second countdown timer
-  Animated timer progress bar
-  Questions are shuffled randomly every time
-  Instant answer validation
-  Correct and incorrect answers highlighted
-  Next button remains disabled until an answer is selected
-  Automatically moves to the next question when time expires
-  Dynamic result screen based on score
-  Restart quiz functionality
-  Fully responsive design

##  How It Works

1. Click **Start Quiz**
2. Read the instructions.
3. Start answering the questions.
4. Each question has one correct answer.
5. Select an answer to unlock the **Next** button.
6. If time runs out, the correct answer is shown automatically.
7. After completing all questions, your final score is displayed.
8. Click **Restart Quiz** to play again.
